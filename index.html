Seguridad Ofensiva & Pentesting

Disciplina que aplica metodologías de ataque controlado (Red Team) para evaluar defensas. Incluye explotación de vulnerabilidades OWASP Top 10, escalada de privilegios, movimiento lateral y exfiltración simulada en entornos Windows/Linux.
Validar controles de seguridad, cumplir normativas (ISO 27001, PCI-DSS), identificar CVEs explotables antes que actores maliciosos, y medir tiempos de detección/respuesta (MTTD/MTTR) del equipo Blue Team.
Siguiendo metodologías como PTES, OSSTMM o MITRE ATT&CK. Herramientas: Burp Suite (web), Metasploit/Cobalt Strike (explotación), Impacket (movimiento lateral), BloodHound (AD), Hashcat/John (cracking), y C2 frameworks para simulación de APTs.

Optimización de Sistemas (Ricing)

Personalización profunda de window managers compostados (Wayland: Hyprland, Sway / X11: i3-gaps, bspwm) con configuración en Lua, Python o bash. Incluye shaders (picom), barras (waybar, polybar) y launchers (wofi, rofi).
Reducir uso de RAM de 2GB (DE tradicional) a ~400MB, mejorar workflow con tiling automático, atajos de teclado personalizados, y crear dotfiles versionados en Git para despliegue rápido en múltiples máquinas.
Mediante archivos de configuración (.config), scripts de autostart, hooks de eventos, temas GTK/Qt, fuentes patched (Nerd Fonts), y herramientas como home-manager (NixOS) o GNU Stow para gestión de dotfiles.

Modding de Dispositivos Android

Modificación del stack de software Android: desbloqueo de bootloader, flasheo de recoveries (TWRP, OrangeFox), root (Magisk, KernelSU, APatch), y Custom ROMs (LineageOS, crDroid, Evolution X). Incluye edición de particiones y kernels.
Revivir dispositivos con bootloader bloqueado, eliminar bloatware de fabricantes, actualizar Android en dispositivos EOL, mejorar batería con kernels undervolteados, y habilitar features como 90/120Hz, VoLTE, o cámaras GCam.
Vía ADB/Fastboot para comandos low-level, desbloqueo OEM, flash de boot/recovery images, modificación de fstab, patch de vbmeta para deshabilitar verity, y uso de módulos Magisk para systemless modifications. Soporte en dispositivos Xiaomi, Pixel, Motorola, OnePlus.

Arch Linux

Distribución Linux rolling release basada en la filosofía KISS (Keep It Simple, Stupid). Actualizaciones continuas sin necesidad de reinstalar, paquete manager pacman ultra-rápido, y acceso al AUR (Arch User Repository) con más de 70,000 paquetes.
Para usuarios avanzados que quieren control total del sistema, software siempre actualizado (últimas versiones el día del release), instalación minimalista desde cero (Arch Install Scripts), y documentación excepcional (Arch Wiki).
Instalación base vía CLI con archinstall o manualmente, configuración de /etc/pacman.conf para repositorios, uso de yay/paru como AUR helpers, systemd para init, y kernels múltiples (linux, linux-lts, linux-zen) según necesidad.

Kali Linux

Distribución Debian-based diseñada para pentesting y auditoría de seguridad. Mantenido por Offensive Security, incluye más de 600 herramientas preinstaladas para testing de penetración, análisis forense, ingeniería inversa y ataques wireless.
Realizar auditorías de seguridad en redes, sistemas y aplicaciones web. Análisis de vulnerabilidades, cracking de contraseñas, sniffing de tráfico, explotación de servicios, y generación de reportes para cumplimiento de normativas.
Herramientas clave: Nmap (escaneo), Metasploit Framework (explotación), Burp Suite (web), Aircrack-ng (WiFi), John the Ripper/Hashcat (password cracking), Wireshark (análisis de paquetes), SQLmap (inyección SQL), y Ghidra/Cutter (reverse engineering).

CachyOS

Distribución basada en Arch Linux optimizada para máxima velocidad y respuesta del sistema. Utiliza compilaciones con flags de optimización avanzados (-O3, LTO, BOLT) y kernels personalizados para reducir latencia.
Para gaming, desarrollo y usuarios que buscan el máximo rendimiento de su hardware, especialmente en procesadores modernos (AMD Zen 4/5, Intel 12ª-14ª gen). Ideal para workstations de producción y gaming de baja latencia.
Utiliza repositorios optimizados con paquetes compilados para x86-64-v3/v4, kernels personalizados (XanMod, Bore, CachyOS kernel con scheduler BORE), sistema de archivos BTRFS con compresión ZSTD por defecto, y caché de paquetes con mirrorselect automático.

PikaOS

Distribución Linux basada en Ubuntu optimizada para gaming y baja latencia. Combina la estabilidad de Ubuntu con kernels de baja latencia y drivers actualizados para máximo rendimiento en juegos y aplicaciones demandantes.
Gaming en Linux con máximo FPS, reducción de input lag, streaming de baja latencia, y ejecución de aplicaciones que requieren respuesta rápida del sistema sin sacrificar compatibilidad con paquetes .deb y PPAs de Ubuntu.
Kernel Linux con patchset de baja latencia (PREEMPT_RT), drivers de video actualizados (Mesa git, NVIDIA latest), governors de CPU performance por defecto, IRQ affinity tuning, y soporte para Wine/Proton optimizado para gaming.

Android Root

Proceso de obtener permisos de superusuario (root) en el sistema operativo Android, permitiendo acceso completo al directorio /system, modificación de archivos del kernel, y ejecución de comandos con privilegios máximos.
Control total del dispositivo: eliminar bloatware del fabricante, overclock/undervolt de CPU/GPU, backup completo con Titanium Backup, automatización avanzada con Tasker, y acceso a aplicaciones que requieren root.
Mediante exploits de kernel o desbloqueo de bootloader + flash de binaries de root (Magisk, KernelSU). Se modifica el boot.img para inyectar el daemon de root, permitiendo gestión de permisos vía apps de superusuario.

Magisk / KernelSU

Magisk: solución de root systemless que no modifica la partición /system. KernelSU: implementación de root a nivel de kernel para dispositivos con GKI (Generic Kernel Image). Ambos permiten ocultar root de aplicaciones bancarias y SafetyNet.
Obtener root sin triplear verificación de integridad, instalar módulos para personalización (Viper4Android, LSPosed, Greenify), y mantener compatibilidad con apps que detectan root mediante MagiskHide o Zygisk.
Magisk: parcheo de boot.img desde la app, flash vía fastboot/ODIN, gestión de módulos desde la interfaz. KernelSU: requiere kernel GKI 5.10+, se instala como módulo del kernel, ofrece mejor aislamiento y performance que soluciones userspace.

Custom ROMs

Sistemas operativos Android modificados por la comunidad que reemplazan el firmware del fabricante. Incluyen LineageOS (sucesor de CyanogenMod), Pixel Experience, crDroid, Evolution X, y ROMs específicas por dispositivo.
Actualizar Android en dispositivos EOL (End of Life), eliminar bloatware, mejorar privacidad con Privacy Guard, optimizar batería y rendimiento, y acceder a la última versión de Android en hardware antiguo.
Desbloqueo de bootloader, instalación de custom recovery (TWRP, OrangeFox), wipe de datos, flash de la ROM + GApps (si no incluye), y opcionalmente Magisk para root. Soporte para dispositivos Xiaomi, Pixel, OnePlus, Motorola, Samsung (Exynos).

Hardware - Optimización & Audio Hi-Fi

Optimización de componentes físicos: curva de optimización para CPUs AMD Ryzen (PBO2, Curve Optimizer), tuning de voltajes, configuración de timings de RAM, y configuración de audio Hi-Fi con DACs externos y reproducción FLAC sin pérdida.
Maximizar rendimiento de CPUs Ryzen con Precision Boost Overdrive 2, reducir temperaturas con undervolt, mejorar estabilidad con overclock manual, y disfrutar de audio de alta fidelidad (24-bit/192kHz) con DACs externos y configuración ALSA/PulseAudio.
BIOS/UEFI: configuración de PBO, Curve Optimizer (negative offsets), FCLK/MCLK tuning. Software: Ryzen Master, HWInfo64, OCCT para estabilidad. Audio: PipeWire/JACK para baja latencia, Tidal/Qobuz para streaming Hi-Res, ecualización con EasyEffects.
